SUBSCRIPTION SYSTEM:

* Plans: Monthly and Yearly (yearly discounted)
* Integrate Stripe Checkout and subscriptions
* Store subscription status in database
* Restrict platform features if subscription inactive
* Handle:

  * renewals
  * cancellations
  * expired subscriptions
* Validate subscription status on every authenticated request

SCORE MANAGEMENT SYSTEM:

* Each user stores exactly 5 latest golf scores
* Each score:

  * integer value between 1–45 (Stableford format)
  * must include a date
* Logic:

  * If a 6th score is added → automatically remove oldest score
  * Always display scores in reverse chronological order (newest first)

DRAW SYSTEM:

* Monthly draw execution (admin-controlled)
* Match tiers:

  * 5-number match (40% prize pool, rollover if no winner)
  * 4-number match (35%)
  * 3-number match (25%)

Draw logic:

* Random number generation (default)
* Optional weighted algorithm based on score frequency
* Admin can:

  * run simulation (preview results)
  * publish final draw

PRIZE POOL LOGIC:

* A portion of each subscription contributes to prize pool
* Automatically calculate pool based on active subscribers
* Split winnings equally among winners in the same tier
* 5-match jackpot rolls over if no winner

CHARITY SYSTEM:

* Users select a charity during signup
* Minimum 10% contribution of subscription fee
* Users can increase percentage voluntarily

Features:

* Charity listing page with search and filtering

* Featured charity section on homepage

WINNER VERIFICATION SYSTEM:
